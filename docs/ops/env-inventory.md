# Environment Inventory

## Runtime

- `FIREBASE_PROJECT_ID`: Firebase / GCP project ID for Admin SDK calls. Default is `nanny-tech`.
- `FIREBASE_STORAGE_BUCKET`: Primary bucket used for attachment writes. Default is `nanny-tech.firebasestorage.app`.
- `NEXT_PUBLIC_APP_ENV`: `production`, `staging`, or `local`.
