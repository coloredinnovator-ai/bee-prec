Bee-Prec is a lawyer-owned cooperative platform for community legal support and civic accountability, delivered as a dedicated website in `/Users/user1/Desktop/BEE-PREC`.

## Product intent (master spec summary)

Bee-Prec is not just a landing page. It is a trust-first digital hub with:

- Lawyer-led consultations and intake workflows.
- Incident reporting for businesses, services, and misconduct.
- Community discussion and peer support channels (a “fan community” layer).
- User profile management with message control, including user-initiated deletion.
- Moderation and audit controls suitable for legal oversight.
- Strong security posture, retention policy, and transparency for legal/ethical governance.

## Next implementation layer (pending)
